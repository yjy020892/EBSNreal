# EBSNreal
Nreal(AR 그래스)를 사용한 원격 수업 테스트 컨텐츠

## 프로젝트 소개
EBS에서 메인기업으로 여러 업체가 붙어서 진행한 컨텐츠

# 개발관련
### 인원
개발1(본인), 아트1, 서버관련 참여업체 1
### 내용
혼자 개발, 기획, 연출 등 모두 구현
### 기술
Nreal AR 글래스 사용

### 실행방법
1. Lobby부터 시작
2. 에디터에서 실행시 컨트롤러 조종 패널에 마우스를 가져다 놓고 Shift로 컨트롤러 이동, 마우스 클릭으로 실행, Space로 시점 이동
3. 교육자는 선생님이 방을 생성해야 입장 가능
